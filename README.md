# Fake Account Detector

This project detects fake social media accounts using Machine Learning and Flask.

## Features
- Detect fake Instagram/Facebook accounts
- Web interface using Flask
- Machine learning model prediction

## Technologies Used
- Python
- Flask
- Machine Learning

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py
